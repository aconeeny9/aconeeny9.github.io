WireFrame Bookshelf

This project was constructed using Balsamiq Cloud. I created a bookshelf website using connected wireframes, loosely inspired by the GoodReads website. It links to multiple different wireframes representing different list-style bookshelves and suggests functionality for adding/removing books to/from shelves and leaving reviews.
